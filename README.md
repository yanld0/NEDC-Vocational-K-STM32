# 全国电子设计大赛职高组K题 - STM32F103C8T6自动避障小车

## 项目信息
- 赛题：全国电子设计大赛职高组K题
- 主控芯片：STM32F103C8T6
- 开发环境：Keil uVision5.38、STM32标准库、Git/GitHub
- 核心功能：实现3种运行模式（任意路径避障/蛇行绕障/绕圈），满足赛事10秒内完成任务要求

## 工程结构
| 目录/文件 | 说明 |
|-----------|------|
| Core/     | 核心源码（main.c、中断服务函数、系统初始化） |
| Drivers/  | 模块化驱动（电机/PWM/红外传感器/按键/定时器） |
| MDK-ARM/  | Keil工程配置文件（可直接编译） |
| Docs/     | 赛题任务书、硬件原理图、设计说明 |
| Output/   | 编译后的.hex文件、功能测试截图 |

## 技术亮点
1. 模块化编程：将电机、传感器、按键拆分为独立模块，降低代码耦合度，提升可维护性；
2. 硬件驱动：基于STM32标准库实现PWM电机调速、红外避障检测、TIM2 ms级计时、按键消抖；
3. 工程化：使用.gitignore过滤无用文件，Git版本管理，代码可追溯、可直接编译烧录。

## 编译与烧录
1. 环境要求：Keil uVision5.38及以上，安装STM32F103C8T6芯片包；
2. 编译步骤：打开MDK-ARM/NEDC-Vocational-K-STM32.uvprojx → 选择芯片 → 点击Build → 生成hex文件；
3. 烧录工具：J-Link/ST-Link，烧录Output/下的hex文件。

## 问题与解决
1. 问题：传感器误判导致小车频繁转向；解决：增加多次检测消抖逻辑，过滤单次干扰；
2. 问题：左右电机转速不一致；解决：校准PWM占空比，实现速度匹配；
3. 问题：Keil工程上传垃圾文件；解决：编写.gitignore，仅保留核心源码和配置。
